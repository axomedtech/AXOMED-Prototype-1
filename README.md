## ✨ Features
- ❤️ Heart Rate Monitoring (BPM)  
- 🌡 Body Temperature Monitoring  
- 🌍 Room Temperature & Humidity Tracking  
- 📊 Real-time Web Dashboard  
- ⚡ ESP32 → Node.js → Web Interface  

---

## 🛠 Hardware
| Component | Description |
|----------|------------|
| ESP32 | Main microcontroller |
| MAX30102 | Heart rate sensor |
| DS18B20 | Body temperature sensor |
| DHT11 | Temperature & humidity |
| OLED | Display output |

---

## 💻 Software
- Arduino IDE  
- Node.js  
- HTML / CSS / JavaScript  

---

## 📡 Architecture

ESP32 → WiFi → Node.js Server → Web Dashboard
